Welcome to 'The most difficult Tetris' implemented with Reinforcement Learning.

## Getting Started
### Install Modules
Windows
```
pip install -r requirements.txt
```

Linux
```
pip3 install -r requirements.txt
```

### Start Application
Train
```
python Train.py
python3 Train.py
```

Tetris A.I and Playable Game Test
```
python Display.py
python3 Display.py
```

Game with trained Model
```
python Demo.py
python3 Demo.py
```
